# IMYOU

IMYOU is a 3D social life simulation game where every character has personality, goals, emotions, reputation and autonomy.

Players can build careers, create companies, study, compete in sports, become influencers and shape their future through meaningful decisions.

## Features

### Open World 3D

* Cities
* Universities
* Hospitals
* Companies
* Shopping centers
* Sports stadiums

### Career System

* Programmer
* Doctor
* Teacher
* Designer
* Athlete
* Entrepreneur
* Influencer
* Lawyer

### Business System

* Create companies
* Hire employees
* Fire employees
* Manage salaries
* Expand operations

### Reputation System

* Professional reputation
* Social reputation
* Trust score

### Artificial Intelligence

* Autonomous NPCs
* Emotions and needs
* Dynamic decision making
* Life goals

## Current Status

Prototype Available

* Character creation
* 3D world
* Career system
* Company management
* Leaderboards
* Persistent database
